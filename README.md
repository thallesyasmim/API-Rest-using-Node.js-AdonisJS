# Adonis API application Inspired By Airbnb

## Getting started

1. Clone this repo using `git clone git@github.com:thallesyasmim/API-AdonisJS.git`
2. Move yourself to the appropriate directory: `cd api-adonisjs`<br />
3. Run `npm install` to install dependencies<br />

### Getting started with the project

1. Run `adonis serve --dev` to start the application on `localhost:3333`
